# نمونه‌ی release note

نسخه‌ی اول نکونیموس از نظر کد و مستندات به مرحله‌ی تست نهایی تلگرام رسیده است.

جریان اصلی ساخت لینک، دریافت پیام ناشناس، inbox، پاسخ، block/report و تنظیم زبان در این نسخه وجود دارد. متن‌های عمومی و مستندات هم با مدل فعلی ticket و retention هماهنگ شده‌اند. چیزی که هنوز باقی مانده، اعمال profile نهایی BotFather و اجرای تست دستی مسیرهای واقعی داخل تلگرام است؛ بنابراین وضعیت دقیق فعلی «آماده برای تست release» است، نه انتشار نهایی.
