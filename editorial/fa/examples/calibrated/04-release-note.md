# نمونه‌ی release note

نسخه‌ی اول Project One از نظر کد و مستندات به مرحله‌ی تست نهایی رسیده است.

جریان اصلی دریافت ورودی، پردازش، پاسخ، مدیریت خطا و تنظیمات پایه در این نسخه وجود دارد. متن‌های عمومی و مستندات هم با مدل فعلی data flow و retention هماهنگ شده‌اند. چیزی که هنوز باقی مانده، اجرای تست دستی مسیرهای واقعی و بررسی نهایی release است؛ بنابراین وضعیت دقیق فعلی «آماده برای تست release» است، نه انتشار نهایی.
