# چک‌لیست هماهنگی فارسی و انگلیسی

- [ ] title و description از نظر معنا هم‌جهت‌اند.
- [ ] featureهای موجود در هر دو نسخه یکسان‌اند.
- [ ] roadmap با feature موجود مخلوط نشده است.
- [ ] security wording هم‌سطح است.
- [ ] dates و versionها هماهنگ‌اند.
- [ ] code و links یکسان یا عمداً متفاوت‌اند.
- [ ] limitation در هیچ نسخه‌ای حذف نشده است.
- [ ] فارسی ترجمه‌ی خط‌به‌خط نیست.
- [ ] اصطلاحات canonical حفظ شده‌اند.
