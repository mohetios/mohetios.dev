# Evaluation

Evalها رفتار موتور را می‌سنجند، نه دانش آن درباره‌ی یک پروژه.

محورها:

- semantic preservation
- factual integrity
- temporal integrity
- scope control
- voice consistency
- natural Persian
- surface adaptation
- context isolation
- protected literal preservation

`cases.json` شامل taskهای generic و substitution-based است. خروجی مدل را در یک JSON مطابق `outputs.example.json` قرار بده و static checker را اجرا کن. ارزیابی معنایی و voice همچنان review انسانی می‌خواهد.
