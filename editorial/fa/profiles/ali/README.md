# Ali Voice Profile

این profile فقط ویژگی‌های نسبتاً پایدار نوشتن و reasoning علی را نگه می‌دارد. اطلاعات پروژه، برنامه‌ی روز، وضعیت شغلی، location، قیمت و roadmap اینجا جایی ندارند.

فایل‌های اصلی:

- `voice.md`
- `reasoning-style.md`
- `register.md`
- `lexicon.md`
- `anti-patterns.md`
- `corpus/`

Corpus تأییدشده از ruleها معتبرتر است، اما فقط برای calibration لحن؛ facts همیشه از source task می‌آیند.
