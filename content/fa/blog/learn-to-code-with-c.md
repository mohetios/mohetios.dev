## یادداشت‌های مطالعه

_Learn to Code with C_ مسیر مبتدی‌دوستی به برنامه‌نویسی از طریق C است. زمینه Raspberry Pi کتاب را عملی می‌کند نه انتزاعی.

مفید است چون C در غیر این صورت یا خیلی آکادمیک می‌شود یا خیلی ترسناک. برنامه‌های کوچک روی ماشین واقعی حلقه بازخورد را ملموس نگه می‌دارند.

## ایده‌های مفید

### برنامه‌های کوچک کافی‌اند

برای یادگیری C، برنامه کوچک کار اسباب‌بازی نیست. جایی است که جزئیات مهم ظاهر می‌شوند: type، شرط، حلقه، تابع، آرایه و ورودی/خروجی.

### زمینه سخت‌افزار کمک می‌کند

زاویه Raspberry Pi کد را به ماشینی وصل می‌کند که خواننده واقعاً می‌تواند اجرا و تغییر دهد.

### اصول قبل از framework

یادآوری است که مهارت برنامه‌نویسی هنوز از ایده‌های کوچک و روشن ساخته می‌شود، قبل از اینکه معماری شود.

## چرا نزدیک نگهش می‌دارم

نقطه شروع خوب برای آموزش، مرور اصول یا گرم‌کردن قبل از کار embedded عمیق‌تر C.
