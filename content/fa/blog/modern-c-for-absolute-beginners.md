## یادداشت‌های مطالعه

_Modern C for Absolute Beginners_ روی قفسه می‌ماند چون C هنوز چیزهایی را آموزش می‌دهد که خیلی از زبان‌های سطح بالاتر پنهان می‌کنند: چیدمان حافظه، کامپایل، pointer، undefined behavior و هزینه هر abstraction.

برای مهندسی امروز، C فقط زبان legacy نیست. راهی است برای ساختن مدل ذهنی دقیق‌تر از کامپیوتر.

## ایده‌های مفید

### حافظه ملموس است

C حافظه را visible می‌کند. متغیر، آرایه، pointer، تخصیص و lifetime از جزئیات مبهم پیاده‌سازی به بخشی از کار تبدیل می‌شوند.

### کامپایل مهم است

مسیر از سورس تا executable را فهمیدن ارزش دارد. header، translation unit، linking و flagهای build همه روی وجود واقعی برنامه اثر می‌گذارند.

### سادگی لبه تیز دارد

C آن‌قدر کوچک است که یاد بگیری، اما آن‌قدر بخشنده نیست که بی‌دقت استفاده شود. این ترکیب معلم خوبی می‌سازد.

## چرا نزدیک نگهش می‌دارم

ورودی خوب برای بازگشت به systems programming بدون اینکه وانمود کنیم هر روز با C زندگی می‌کنیم.
