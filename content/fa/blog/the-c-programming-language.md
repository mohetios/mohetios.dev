## یادداشت‌های مطالعه

_The C Programming Language_ هنوز ارزش دارد که نزدیک باشد چون فشرده، مستقیم و به‌طرز غیرعادی روشن است. سعی نمی‌کند C بزرگ‌تر از آنچه هست احساس شود.

ارزش ماندگارش سبک فکری است که تشویق می‌کند: برنامه کوچک، زبان دقیق، interface مراقب و احترام به کاری که ماشین انجام می‌دهد.

## ایده‌های مفید

### وضوح از حجم جلو می‌زند

کتاب کوتاه است چون توضیح‌ها منضبط‌اند. کنتراست مفیدی با کتاب‌های برنامه‌نویسی است که بدون دقت بزرگ می‌شوند.

### C به فکر دقیق پاداش می‌دهد

C از خیلی اشتباهات محافظت نمی‌کند. فکر دقیق بخشی از زبان می‌شود.

### interface می‌تواند کوچک باشد

خیلی مثال‌ها نشان می‌دهند با تابع کوچک و جریان داده روشن چقدر می‌شود انجام داد.

## چرا نزدیک نگهش می‌دارم

مرجع C که می‌خواهم وقتی بعد از مدتی برمی‌گردم در دسترس باشد. هم تاریخی است هم هنوز عملی.
