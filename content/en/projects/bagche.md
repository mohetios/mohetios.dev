Bagche is a product workspace around commerce, publishing, and lightweight web infrastructure. The public repositories currently show two visible surfaces: the Bagche front page and Mamoochi, an open-source Jamstack framework built with Nuxt, Vue, and Cloudflare-friendly assumptions.

This project page should stay in review until the boundary between public product story, private experiments, and organization history is clear.

Repositories:

- [bagche/mamoochi](https://github.com/bagche/mamoochi)
- [bagche/Front](https://github.com/bagche/Front)

## What To Preserve

The interesting thread is not one repository by itself. It is the move toward small, composable product infrastructure:

- a public-facing organization surface,
- an open-source Jamstack framework through Mamoochi,
- Cloudflare-friendly deployment,
- Vue and Nuxt as the application layer,
- room for future commerce experiments without committing to a heavy platform too early.

Bagche is worth documenting because it connects product naming, public web presence, and reusable software foundations. That is a different story from a library or a one-off app.
